The universal selector * sets all text color to bold.

All elements containing the class "warning" have an orange background.

This is due to the "*.warning" selector. In cases like this, the "*" can be omitted.

For example, this paragraph is of the class "danger". It is being colored red by ".danger", without the universal selector at the front.

Now, let's apply some nesting. All the <span> elements within this div will be underlined . It does not matter how deep it is, or in which element it is contained. This is due to the "div * span" selector. This means it selects all <span> that are a descendant of a <div>, with at least 1 element in between

The reason we didn't do "div span" is that we do not want direct children of this div be underlined as well.

For example, "this" is a direct child of <div>, wrapped in <span> tags, but it is not underlined!
